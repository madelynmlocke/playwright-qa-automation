# Test Case: Profile Navigation Link

## Test ID
NAV-001

## Test Title
Verify that the `Profile` navigation link redirects the user to the Profile page.

## Test Type
Functional / Navigation Test

## Preconditions
- User is logged into the application
- Navigation bar is visible

## Test Steps
1. Open the application homepage.
2. Locate the `Profile` link in the main navigation bar.
3. Click the `Profile` link.

## Expected Result
- The user is redirected to the **Profile page**.
- The URL updates to `/profile`.
- The Profile page heading or content is displayed.
